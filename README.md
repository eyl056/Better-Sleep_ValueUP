# Better-Sleep_ValueUP
> 알람 + 수면 상태 분석, 기록



## 프로젝트 내용
* 스마트폰에 내장되 자이로 센서와 가속도 센서를 이용하여 얻은 사용자의 수면 여부를 유추할 수 있는 데이터에 따라 조명과 알람을 자동적으로 제어
* 아두이노 블루투스를 활용해 직접 개발한 스마트폰 어플과 조명을 연동하고 아두이노 센서를 사용해 사람의 수면 상태 측정

## 개발 환경 / 언어
```
 * Arduino
 * Android Studio
 * Java
 * C
 
```
